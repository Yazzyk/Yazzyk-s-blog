Mysql连接失败(61)

2020-03-19 PaleBlueYk

0x01

这个问题已经存在很久了,首先排除掉一些已存在的可能性:

  • 用户没有远程登录权限
  • 端口不是3306
  • 配置文件中注释bind-address

0x02

排除以上信息后,我在一片文章中找到答案MySQL 连接错误Can’t connect to MySQL server on ' ‘(61)
由于我重置了服务器系统,升级为了centos7所以忘记了防火墙的设置

➜  ~ sudo firewall-cmd --zone=public --permanent --add-service=mysql
success
➜  ~ sudo systemctl restart firewalld

重启防火墙后,Nacicat连接成功!
